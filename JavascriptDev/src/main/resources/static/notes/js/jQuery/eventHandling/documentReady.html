document ready()

ready()
-------
jQuery fires a custom event named 'ready' when the DOM is loaded and available for manipulation.
You can attach as many ready() events to the document as you would like.
They are executed in the order they were added top down in script tag.
FYI: ready() triggers before CSS is loaded. After jQuery 1.3, always include all CSS files before any jQuery code.
FYI: ready() triggers as script is parsed, not after.
onload()
--------
jQuery window.onload event loads after all assets including all scripts are loaded.
