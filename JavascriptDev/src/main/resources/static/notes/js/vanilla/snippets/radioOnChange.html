Radios in groups

Notice the 3 different groups of radios written in different HTML structures for various effects.

allotRadios

Theses two radios sharing the same value for the name attribute ("allotRadios")
configures them so that only one radio of the two can be checked at any time.


transferRadios

See how selecting one of the two radios above does NOT uncheck this radio - that's
because this radio's name attribute equals "transferRadios", not "allotRadios".

catsRadio

All of these radios have no name attribute - since they're not grouped, they can
all be selected at the same time. Consider using checkboxes instead for this effect.
