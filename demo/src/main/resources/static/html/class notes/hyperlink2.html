12強／日本隊最大假想敵台灣 日媒警戒「二刀流」劉致榮

為備戰11月開始的12強棒球賽，各隊已經緊急加強集訓，其中又以B組日本、台灣、委內瑞拉、波多黎各最為激烈，
尤其是中華隊也被日本媒體《棒球頻道》點名為最大強敵。為備戰11月開始的12強棒球賽，各隊已經緊急加強集訓，其中又以B組日本、台灣、委內瑞拉、波多黎各最為激烈，尤其是中華隊也被日本媒體《棒球頻道》點名為最大強敵。

中華隊陣容由旅美、旅日還有中職好手所組成，其中江少慶、胡智為、張奕、吳昇峰為主先發陣容，江少慶可說是王牌，本季在3Ａ也有9勝，胡智為過去也有大聯盟經驗。

此外，林立、陳俊秀、朱育賢、林泓育等打者也都擁有不錯時實力，也是Lamigo奪下三連霸的主軸，加上旅日好手王柏融，整體打擊實力也被評價很高。

20歲的「二刀流」火球男劉致榮更被注目，不但他在亞錦賽中華隊幫助台灣拿下睽違18年冠軍，最快也飆出157公里火球，之後也跟紅襪隊簽下合約。
