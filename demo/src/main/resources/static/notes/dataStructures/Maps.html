Javascript Maps

Console

Create a Map

                let map = new Map();


Add/Remove key-value pairs from Map

                // Add pairs
                map.set('1', 'str');
                map.set(1, 'num');
                map.set(true, 'bool');
                


                // Remove pairs
                map.delete('1', 'str');
                map.delete(1, 'num');
                map.delete(true, 'bool');


Get a value by providing key from Map

                // Get a value from key-value pairs in a Map
                map.get('1');       // 'str'
                map.get(1);         // 'num'
                map.get(true);      // 'bool'
                map.get('aLife');   // ??
